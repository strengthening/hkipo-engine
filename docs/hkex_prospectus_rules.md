# 港股招股书法定披露要求

## 监管框架

三层规则叠加：
- **CWUMPO 第三附表** — 法定内容要求
- **HKEX 上市规则第 11 章 + Part 1** — 强制披露要求
- **附录 1-A (Appendix 1-A)** — 具体列出招股书必须包含的每一项内容

## 法定必须披露的信息

对 agent 做打新决策最有价值的字段标 ★：

### 发行信息
- ★ 发售价格/价格区间
- ★ 认购期（公开发售起止日期）
- ★ 发售股份数量及结构（国际配售/公开发售比例）
- ★ 募资总额及用途 (Use of Proceeds)
- 上市方式（全球发售、配售、介绍上市等）

### 公司基本面
- ★ 业务性质及描述
- ★ 历史及发展沿革
- ★ 行业概览及竞争格局
- 董事及高管信息
- 控股股东及持股结构
- ★ 基石投资者及锁定期

### 财务信息
- ★ 过去三个完整财年的经审计财报
- ★ 如最近财年结束超过 6 个月，须附额外中期报表
- ★ 管理层讨论与分析 (MD&A)
- 每股净有形资产值
- ★ 盈利预测（如有）

### 风险及合规
- ★ 风险因素（每项聚焦单一主要风险）
- 监管概览及适用法律
- 上市规则豁免/减免
- 公司治理信息

### 其他法定报告
- 物业估值报告（如适用）
- 技术报告（如矿业公司）
- 重大合同摘要

## 上市方式说明

| 方式 | 是否发新股 | 是否公开认购 | 特点 |
|------|-----------|-------------|------|
| 全球发售 | 是 | 是 | 最常见，分国际配售 + 公开发售，散户能打新 |
| 配售 | 是 | 否 | 只卖给机构/专业投资者，散户参与不了 |
| 介绍上市 | 否 | 否 | 不融资，只挂牌交易（如从其他交易所转来） |

> agent 打新只涉及"全球发售"方式，其余方式标记即可。
